Project Setup Guide

This project consists of a backend and a frontend, each running separately.
Follow the steps below to install dependencies and start the development environment.

1 Backend Setup

Open the project and navigate to the backend directory:

cd backend

Install all required dependencies:

npm install

Start the development server:

npm run dev

The backend is already configured to use a cloud MongoDB Atlas database, so no additional setup is required.

1 Frontend Setup

Open a second terminal and navigate to the frontend directory:

cd frontend

Install dependencies:

npm install

During installation, you may see the prompt:

✔️ Are you interested in participating?

Simply press Enter to continue.

Start the development server:

npm run dev

The application will be available at:

http://localhost:3000
